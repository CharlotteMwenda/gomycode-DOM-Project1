Shopping Cart Project
📌 Overview

This project is a fully functional shopping cart interface built using JavaScript (DOM manipulation), along with the provided HTML and CSS. It simulates a real-world e-commerce cart where users can interact with items dynamically.

🎯 Objectives

The goal of this project is to implement JavaScript functionality that allows users to:

Adjust item quantities
Delete items from the cart
Like/unlike items
View the total price updating in real-time
⚙️ Features
➕➖ Quantity Adjustment
Each item has “+” and “−” buttons
Users can increase or decrease item quantity
Quantity updates instantly on the UI
🗑️ Delete Items
Each product has a delete button
Clicking it removes the item from the cart
The total price updates automatically
❤️ Like Items
Each item has a heart icon
Clicking toggles the “liked” state
The icon changes color (e.g., grey → red)
💰 Total Price Calculation
The total price updates dynamically based on:
Item quantity changes
Item deletions
Ensures accurate cart value at all times
🧠 Concepts Used
JavaScript DOM Manipulation
Event Listeners (click events)
Traversing the DOM
Updating UI dynamically
Basic arithmetic operations
🛠️ Technologies
HTML (provided)
CSS (provided)
JavaScript (custom logic)
